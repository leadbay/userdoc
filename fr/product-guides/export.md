# Export

Exportez des leads depuis n'importe quel onglet pour les utiliser en dehors de Leadbay.

---

## Comment exporter

1. Sélectionnez des leads avec les cases à cocher
2. Cliquez sur **Export** dans la barre d'outils
3. Choisissez votre format

### Export CSV

Deux fichiers sont générés :

- **Entreprises** : nom, description, localisation, secteur, taille, site web, SIREN, statut
- **Contacts enrichis** : nom, prénom, email, téléphone, poste, profil LinkedIn

Seuls les contacts qui ont été enrichis sont inclus dans le fichier contacts.

### Export carte

Sélectionnez des leads et cliquez sur **Map** pour les visualiser géographiquement. Vous pouvez :

- Naviguer entre les emplacements des leads
- Partager le lien avec des collègues
- Ouvrir la carte sur votre téléphone pour les visites terrain

---

## Que se passe-t-il après l'export ?

- Les leads exportés depuis **Discover** passent dans **Monitor** — c'est normal, ils intègrent votre pipeline.
- Les leads dans **Monitor** et **Activate** restent en place après export.
- Les données exportées peuvent aussi être envoyées automatiquement vers votre CRM via [Zapier](data-sources.md).

{% hint style="info" %}
Si des leads semblent « disparaître » de Discover après export, vérifiez Monitor — c'est là qu'ils se trouvent.
{% endhint %}
