# Premiers pas

{% hint style="info" %}
L'onboarding n'est nécessaire que la première fois que votre organisation rejoint Leadbay. Si vous êtes invité dans une organisation existante, vous pouvez commencer à utiliser Leadbay immédiatement.

Inscrivez-vous sur [https://leadbay.ai/](https://leadbay.ai/) — pas de démo, commencez gratuitement.
{% endhint %}

Lors de votre première inscription, Leadbay vous guide à travers un **onboarding en 8 étapes** qui présente les fonctionnalités clés de manière pratique.

## Étape 1 — Créer votre compte

Nous vous demanderons votre nom, email et un mot de passe. Vous vérifierez ensuite votre email en cliquant sur un lien que nous vous envoyons.

## Étape 2 — Parlez-nous de vos affaires passées

Pour personnaliser les recommandations, notre IA a besoin d'un aperçu de vos leads passés.

### Uploader un CSV

Fournissez un fichier CSV avec vos données CRM passées. Il n'a pas besoin d'être parfait ni complet — plus il est riche, mieux c'est, mais nous travaillons avec ce que vous nous donnez.

<figure><img src="../../.gitbook/assets/Capture d'écran 2025-08-20 à 10.50.22.png" alt="" width="375"><figcaption></figcaption></figure>

### Mapper les champs

Chaque CRM nomme les choses différemment. Leadbay détecte automatiquement vos colonnes — vérifiez ou corrigez le mapping.

<figure><img src="../../.gitbook/assets/Capture d'écran 2025-08-20 à 10.51.00.png" alt="" width="375"><figcaption></figcaption></figure>

Le champ le plus important est **Status** — il indique à notre IA quels leads ont été gagnés ou perdus.

<figure><img src="../../.gitbook/assets/Capture d'écran 2025-08-20 à 10.51.36.png" alt="" width="375"><figcaption></figcaption></figure>

Une fois terminé, vous verrez votre première sélection de leads recommandés par l'IA.

{% hint style="info" %}
Pas de CSV ? Vous pouvez quand même commencer — Leadbay construira des recommandations à partir de votre description et de vos questions de qualification. Vous pourrez toujours importer des données plus tard depuis [Sources de données](data-sources.md).
{% endhint %}
